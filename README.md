# EndlessMuseum

Add a new museum display room of arbitrary size, always large enough to fit everything you need to donate.
The new display room's decor changes randomly, it might be haunted.

NOTE: This room does not appear if you do not require more donate slots than you have!

Aside from the display room, this mod also makes sure your donated artifacts are not left out of bounds.

## Installation

1. Download and install SMAPI.
2. Download this mod and install to the Mods folder.

## Configuration

- `Enable Glass`: Add glass case to the new displays similar to SVE

- `Base Origin`: Top left position of the new museum room, calculated from the bottom edge of the map

- `Door Position`: Where to put door that goes to the new room

- `Min Room Width`: Minimum width of the room, used if this is greater than the map's width

- `Min Slot Count`: Ensure that the display room has at least this many slots
